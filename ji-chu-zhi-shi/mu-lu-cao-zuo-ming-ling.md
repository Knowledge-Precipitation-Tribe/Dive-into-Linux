# 目录操作命令

## 切换目录

```text
cd [目录]
```

### 相对路径

| 标识符 | 含义 |
| :--- | :--- |
| ~ | 当前用户家目录 |
| - | 上一次的目录 |
| . | 当前目录 |
| .. | 上一级目录 |

### 绝对路径

指从跟目录开始一级一级递归查找的形式，如下所示。优点是无论在任何位置都能直接切换到指定的目录下。

```text
/root/data
```

## 创建目录

```bash
mkdir [-p] [目录名]
```

其中-p表示递归创建

## 删除目录

```bash
rmdir [-f] [目录名]
```

其中-f可表示强制删除

## 查询目录内容

```bash
ls [-aldhi] [文件]
```
